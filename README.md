# Névora

Metroidvania 2D de ação e exploração, **cooperativo (1–4 jogadores)**, com
identidade visual e narrativa 100% originais. Plataforma inicial: **Web +
PWA**. Futuro: Steam, Epic Games Store, Google Play, App Store.

> 🟡 Projeto em **Fase 0 — Pré-produção**. Nenhum código de jogo ainda: este
> repositório concentra o planejamento e a documentação completa antes do
> desenvolvimento começar.

## Comece por aqui

Toda a documentação do projeto está em [`docs/`](./docs/README.md):
visão geral, enredo completo, mundo e biomas, gameplay, multiplayer,
tecnologia, arte/áudio, publicação e roadmap.
